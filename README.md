# The Frontier AI Briefing — Executive Program (May 2026)

A ten-module, in-browser executive program on the strategy, architecture, economics, governance, and operationalization of agentic AI in 2026. Designed in the editorial register of MIT Sloan Executive Education and McKinsey learning materials — written for senior leaders (CIO / CFO / COO / board), not for first-time users.

## Play it

**Live:** https://ram1729.github.io/AILearningGamev0.1/

Or open `index.html` directly in any modern browser — same file, no setup.

- No build step
- No external dependencies (vanilla HTML / CSS / JavaScript, system fonts only)
- No API calls (zero cost, fully offline-capable)
- Progress persisted to `localStorage`
- Approximately 60 minutes, self-paced

## Curriculum

| | Module | Subject |
|---|---|---|
| I    | Foundations                          | Defining frontier AI; the four-provider landscape; the Jagged Frontier and workforce planning |
| II   | Trajectory & Lexicon                 | The five breakthrough moments (ImageNet → agentic pivot); architectural primitives (foundation models, context window, MoE, few-shot) |
| III  | Strategic Inquiry                    | The cost of vague prompting; refining a strategic question |
| IV   | Prompt Architecture                  | Five techniques for decision-grade output; capstone diagnosis |
| V    | Capability Selection                 | Choosing the right system — Opus 4.7, GPT-5.5, Gemini 3.1 Pro, Copilot; reading benchmarks (MMLU saturation, GDPval) |
| VI   | The Economics of AI                  | The BCG 10-20-70 rule; sovereign AI and supply-chain risk |
| VII  | Persistent Programs & Connectivity   | Projects, Custom GPTs, Gems, Agent 365; the Model Context Protocol |
| VIII | The Agentic Pivot                    | Sequencing the build portfolio; three levels of autonomy; the strategic threshold of an "agent" |
| IX   | Governance & Risk                    | Five pillars of agentic governance; sycophancy and executive decision quality |
| X    | The Executive Playbook               | The MIT Sloan five working rules — the closing capstone |

Each module unlocks on completion of the previous one. Distinctions are awarded per module.

## Design notes

The visual treatment uses a serif headline face (Iowan Old Style → Palatino → Georgia fallback), restrained ink-on-cream palette, navy accent, and gold for editorial highlights. No emoji in chrome. Sharp 2px corners, generous whitespace, small-caps eyebrows.

Content is written in the register expected of an executive audience — board-level scenarios, capital-allocation framing, vendor-selection logic, and governance implications.

## Sources

Synthesized from the MIT Sloan "Artificial Intelligence: Implications for Business Strategy" curriculum, BCG's enterprise AI research, Gartner forecasts, Stanford HAI's work on the Jagged Frontier, and the major frontier provider documentation as of May 2026.
