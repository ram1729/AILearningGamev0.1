# Prompt Quest — AI Learning Game v0.1

A single-file, in-browser RPG that teaches AI fundamentals for 2026 — basics, prompt engineering, model comparison, custom skills, and agents.

## Play it

**Live:** https://ram1729.github.io/AILearningGamev0.1/

Or just open `index.html` in your browser locally — it's the same file.

- No build step
- No dependencies (vanilla HTML / CSS / JavaScript)
- No API calls (zero cost, fully offline)
- Progress is saved to `localStorage`

## What's inside

Six worlds, each with story-driven quests:

1. The Beginner's Village — what AI is in 2026
2. The Whispering Forest — how to talk to AI
3. The Engineer's Forge — prompt engineering techniques
4. The Comparison Colosseum — picking the right model (Opus 4.7 / GPT-5.5 / Gemini 3.1 Pro / Copilot)
5. The Skill Sanctum — Projects, Custom GPTs, Gems, Agents
6. The Artifact Tower — building real things and the agentic shift

XP, levels, badges, and a daily streak track your progress.
