# The Frontier AI Briefing — Executive Program (May 2026)

A six-module, in-browser executive program on the strategic, technical, and operational landscape of agentic AI in 2026. Designed in the editorial register of MIT Sloan Executive Education and McKinsey learning materials — written for senior leaders (CIO / CFO / COO / board), not for first-time users.

## Play it

**Live:** https://ram1729.github.io/AILearningGamev0.1/

Or open `index.html` directly in any modern browser — same file, no setup.

- No build step
- No external dependencies (vanilla HTML / CSS / JavaScript, system fonts only)
- No API calls (zero cost, fully offline-capable)
- Progress persisted to `localStorage`

## Curriculum

| | Module | Subject |
|---|---|---|
| I   | Foundations             | Defining frontier AI for the boardroom; the four-provider landscape |
| II  | Strategic Inquiry       | The cost of vague prompting; refining a strategic question |
| III | Prompt Architecture     | Five techniques for decision-grade output; capstone diagnosis |
| IV  | Capability Selection    | Choosing the right system — Opus 4.7, GPT-5.5, Gemini 3.1 Pro, Copilot |
| V   | Persistent Programs     | Projects, Custom GPTs, Gems, and Agent 365 as enterprise primitives |
| VI  | Operationalization      | Sequencing the build portfolio; the strategic threshold of an "agent" |

Each module unlocks on completion of the previous one. Distinctions are awarded per module.

## Design notes

The visual treatment uses a serif headline face (Iowan Old Style → Palatino → Georgia fallback), restrained ink-on-cream palette, navy accent, and gold for editorial highlights. No emoji in chrome. Sharp 2px corners, generous whitespace, small-caps eyebrows.

Content is written in the register expected of an executive audience — board-level scenarios, capital-allocation framing, vendor-selection logic, and governance implications.
